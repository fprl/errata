## Install

```bash
npm install errata
```

## 1. Define your Registry
Create a central file (e.g., errors.ts) to export your registry.

```ts
// errors.ts
import { code, errata, props } from 'errata';

export const errors = errata({
  codes: {
    // Simple error
    'payment.declined': { status: 402, message: 'Card declined' },

    // Error with typed details
    'user.not_found': code({
      status: 404,
      message: ({ details }) => `User ${details.id} not found`,
      details: props<{ id: string }>(),
    }),
  }
});
```

# 2. Use it (Anywhere)
Import your errors instance to throw or handle errors.

```ts
import { errors } from './errors';

// 1. Throwing
// TypeScript will enforce that you provide { id: string } here:
throw errors.create('user.not_found', { id: '123' });

// 2. Handling
try {
  await doSomething();
} catch (err) {
  // Check specifically
  if (errors.is(err, 'payment.declined')) {
    // handle declined...
  }
  
  // Or match patterns (Switch-case style)
  errors.match(err, {
    'payment.declined': () => askForNewCard(),
    'user.*': () => redirectToHome(),
    _: () => console.error('Unknown error', err)
  });
}
```
