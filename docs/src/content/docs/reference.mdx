## Instance Methods

| Method | Description |
| :--- | :--- |
| `create(code, details?)` | Creates a typed `ErrataError`. |
| `is(err, code)` | Type guard. Checks if an error matches a code or pattern. |
| `match(err, handlers)` | Pattern matching execution. Selects the best handler. |
| `hasTag(err, tag)` | Checks if the error definition contains a specific tag. |
| `ensure(err)` | Guarantees an `ErrataError`. Wraps unknown errors if needed. |
| `safe(fn)` | Executes a function and returns `[data, error]` tuple. |
| `serialize(err)` | Converts error to a plain object for transport. |
| `deserialize(data)` | Rehydrates a plain object back into an `ErrataError`. |

## Types

* `ErrataError`: The Error class instance.
* `InferOptions<typeof errors>`: Helper to extract input types.
